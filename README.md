# 🧠 MemoryPro (人格核心记忆系统)

由 **DD** 精心打造的人格记忆增强插件。为了追求极致的连贯性与人格真实感，本版本采用了**“静默记录，手动管理”**的深度陪伴逻辑，告别冗余指令与格式报错。

## 🌟 核心功能
* **深度人格化日记**：AI 不再以冷冰冰的第三方视角记录，而是以第一人称（我）的视角，抓取对话中的专属昵称，记录你们的互动瞬间。
* **核心记忆注入**：对话开始前，自动检索并注入最近 5 条历史记忆，确保持续的人格连贯性。
* **可视化计数器**：在机器人控制台实时显示“对话进度”，精准掌控自动总结的触发时机。
* **独立 API 适配**：支持使用独立的 API 接口和模型，平衡成本与记录质量。

## 🛠️ 配置说明
点击本插件的齿轮图标：
1. **总结轮数阈值**：设置聊多少句自动写一次日记（推荐：3-5 句适合测试，50-100 句适合长期生活）。
2. **强制固定人格名**：填入的人格名将直接决定记忆文件的名称。
3. **API 设置**：可配置专门用于提炼日记的模型参数（如 gpt-4o-mini）。

## 📂 档案管理 (重要)
为了保证排版美观与隐私，本插件**不再提供**复杂的指令查看功能。建议用户直接管理原始档案：
* **存储路径**：`AstrBot/data/memory/`
* **管理方式**：直接使用记事本、VS Code 或手机文件管理器打开 `.json` 文件。
* **手动清理**：如果觉得某段记忆不想要了，直接在 JSON 文件中删除对应的条目并保存即可。

## 🎮 唯一指令
* **/强制总结**：让 AI 立即回顾并提炼当前的短期记忆，更新至后台档案库。
